# 🌦️ WeatherNow

A beautiful **weather app** with glassmorphism UI, animated icons, 5-day forecast, and geolocation — built with pure Vanilla JavaScript.

## ✨ Features

- 🔍 **City search** — look up any city worldwide
- 📅 **5-day forecast** — daily weather at a glance
- 💧 Humidity, 💨 Wind speed, 🌡️ Feels like
- 🌙 **Glassmorphism dark UI** — stunning visual design
- ⚡ **Demo mode** — works even without an API key
- ⌨️ Press **Enter** to search

## 🛠️ Tech Stack

- **Vanilla JavaScript** — zero frameworks
- **OpenWeather API** — free weather data
- **CSS Glassmorphism** — backdrop-filter, gradients
- Single HTML file — just open and use

## 🚀 Setup

1. Get a free API key at [openweathermap.org](https://openweathermap.org/api)
2. Open `index.html` and replace `YOUR_OPENWEATHER_API_KEY` with your key
3. Open the file in your browser — done!

```bash
# Or serve it locally
npx serve .
```
